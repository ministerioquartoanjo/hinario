# Manual de Vídeos para Hinário

## Como Encontrar Vídeos Reais com Legendas

### Passo 1: Verificar os Vídeos Atuais
Para cada hino, verifique os URLs adicionados:

1. Abra o link do YouTube
2. Verifique se o vídeo corresponde à letra do hino
3. Confirme se há legendas (CC) disponíveis:
   - Clique no ícone ⚙️ (Configurações)
   - Selecione "Legendas/CC"
   - Verifique se "Português" está disponível

### Passo 2: Buscar Vídeos Alternativos
Se o vídeo atual não for adequado:

1. **Use termos de busca específicos:**
   - "Título do Hino" + "hino letra"
   - "Título do Hino" + "hinário"
   - "Primeira linha da letra" + "hino"

2. **Prefira canais oficiais:**
   - Igrejas Adventistas
   - Corais conhecidos
   - Canais de música cristã

3. **Verifique a qualidade:**
   - Áudio claro
   - Letra correta
   - Legendas funcionais

### Passo 3: Atualizar o Arquivo
Para substituir um vídeo:

```javascript
"videos": [
    "https://www.youtube.com/watch?v=NOVO_ID_AQUI",
    "https://www.youtube.com/watch?v=OUTRO_VIDEO"
]
```
